# UI/UX Design Principles

## Overview
Guides the creation of premium-feeling user interfaces. Moves beyond generic layouts to implement clean typography, asymmetry, and micro-animations.

## When to Use
- Reviewing dashboard components.
- Designing landing pages.
- Building custom UI elements.

## Instructions
1. Establish clear visual hierarchy using typography sizing and weights.
2. Use harmonious colors and dark/light modes.
3. Ensure generous, consistent padding and margin spacing.
4. Add interactive hover states and smooth animations.

# Minimum Lovable Product (MLP)

## Overview
Focuses on building MVP iterations that are not just barely functional, but high-quality and delightful to use.

## When to Use
- Designing the initial version of a feature.
- Narrowing down product scope for launch.
- Reviewing UX polish and craft.

## Instructions
1. Identify the core user value loop.
2. Cut secondary dashboard features, configurations, and complex settings.
3. Polish the core loop UI/UX to create a highly premium, fast, and satisfying experience.
4. Ship fast and collect feedback immediately.

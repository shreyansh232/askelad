# Startup Metrics Framework

## Overview
Establishes the primary and secondary metrics that measure startup growth, product market fit, and unit economic health.

## When to Use
- Setting up analytics dashboards (Mixpanel, PostHog).
- Evaluating overall startup performance.
- Preparing investor updates.

## Instructions
1. Define one single North Star Metric (NSM) representing core user value (e.g. active daily operations).
2. Measure retention using cohorts (Weekly/Monthly retention curves).
3. Track key SaaS variables: MRR, Churn rate, LTV, CAC, and Burn Multiple.

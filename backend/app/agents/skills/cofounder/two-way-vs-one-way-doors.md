# Two-Way vs. One-Way Doors Decision Model

## Overview
Speed up decision making by dividing options into irreversible, high-stakes decisions (One-Way Doors) and reversible, low-stakes decisions (Two-Way Doors).

## When to Use
- Prioritizing daily operations.
- Selecting technology stacks or core architectural frameworks.
- Hiring early employees or setting equity distributions.

## Instructions
1. Check if the decision is easily reversible without significant resource loss (Two-Way Door).
2. If it is a Two-Way Door, default to speed: make the decision fast and adjust later.
3. If it is a One-Way Door, delay the decision to perform thorough research, build consensus, and analyze risks.

# Opportunity Cost Analysis

## Overview
Every decision to build Feature A is a decision NOT to build Feature B. This skill guides founders to analyze the hidden cost of developer time, focus, and resources.

## When to Use
- When deciding between two high-priority roadmap features.
- When debating whether to build in-house or buy third-party software.
- When selecting customer acquisition channels.

## Instructions
1. Calculate the direct resources (time, money) required for the primary choice.
2. Identify the best alternative choice that will be foregone.
3. Quantify the expected value (revenue, retention, efficiency) of the alternative.
4. Compare values and decide only if the primary option significantly outperforms the alternative.

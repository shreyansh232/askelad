# Pricing Strategy

## Overview
Guides the creation and optimization of pricing structures. Moves beyond cost-plus pricing to design value-based, expansion-friendly pricing tiers.

## When to Use
- Launching an MVP and setting initial prices.
- Redesigning pricing tiers to improve average contract value (ACV) or Net Revenue Retention (NRR).
- Evaluating usage-based vs flat-rate pricing.

## Instructions
1. Identify the primary value metric (what drives value for the user - e.g., contacts, bandwidth, transactions).
2. Segment users into 3 distinct buyer personas (e.g. Hobbyist, Growth, Scale).
3. Set pricing tiers corresponding to these personas.
4. Establish clear upgrade triggers between tiers.

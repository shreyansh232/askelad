# Cloud and Infrastructure Cost Optimization

## Overview
Perform audits on third-party SaaS subscriptions, APIs, database instances, and hosting services to minimize infrastructure costs.

## When to Use
- When monthly hosting bills spike unexpectedly.
- Setting up the initial production database and server architecture.
- Reviewing software tool spend.

## Instructions
1. Audit database indexes and queries to reduce CPU usage.
2. Select right-sized server and container instances (avoid over-provisioning).
3. Set up spending alerts on AWS, GCP, Vercel, Supabase, and OpenAI.
4. Clean up unused development environments, backups, and staging services.

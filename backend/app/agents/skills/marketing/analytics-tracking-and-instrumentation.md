# Analytics Tracking and Instrumentation

## Overview
Guides the setup and management of client-side and server-side event tracking to understand user flow and capture dropoff points.

## When to Use
- Implementing PostHog, Mixpanel, or Google Analytics.
- Measuring landing page signup conversions.
- Tracking retention cohort engagement.

## Instructions
1. Map key funnel stages: Page View -> Sign Up -> Activation Event -> Retention Event.
2. Implement auto-capture or explicit custom event tracking for primary CTAs.
3. Verify event payloads contain user attributes for segmentation.
4. Set up conversion funnel dashboards to locate dropoff points.

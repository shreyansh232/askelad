# Conversion Rate Optimization (CRO)

## Overview
Optimize conversion rate across the landing page, sign-up flow, and activation funnel. Focus on reducing onboarding friction.

## When to Use
- Reviewing page bounce rates.
- Optimizing user sign-up or checkout forms.
- Designing landing pages.

## Instructions
1. Audit landing page layout: ensure the call-to-action (CTA) is visible above the fold.
2. Remove optional form fields during registration to reduce friction.
3. Integrate social proof (testimonials, logos, case studies) and clear security badges.
4. Set up A/B testing on primary headlines and CTAs.
